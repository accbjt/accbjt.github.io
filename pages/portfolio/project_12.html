Restaurant Voting Application

This was a team project done at the Coding House. The purpose of this project was to build a web application using the MEAN Stack (Mongodb, Express.js, Angular.js, and Node.js). This was also a way for us to learn Jira project management software and to use Github as a team. The app a lunch voting application, where you can vote for a restaurant to eat for lunch. Then everyone orders there item from the voted restaurant menu. This application uses sign in, user sessions, administrative access, making Push, Post, Get, and Delete requests.
